# Defensa y Asalto a la Base

## Descripción

Defensa y Asalto a la Base es un juego estratégico desarrollado en Python utilizando programación orientada a objetos y la biblioteca Tkinter para la interfaz gráfica.

Dos jugadores participan en una partida. Uno asume el rol de defensor y el otro el de atacante. El defensor construye estructuras defensivas para proteger su base, mientras que el atacante despliega unidades con el objetivo de destruirla.

El juego incluye sistema de registro e inicio de sesión, selección de roles, selección de facciones, sistema de rondas, ranking de jugadores y almacenamiento de datos mediante archivos JSON.

## Características

* Registro de usuarios.
* Inicio de sesión.
* Ranking de jugadores.
* Selección de roles.
* Selección de facciones.
* Sistema de economía.
* Sistema de combate.
* Sistema de rondas.
* Interfaz gráfica con Tkinter.
* Persistencia de datos mediante JSON.
